# bookshop
-
Установка и запуск

1. Скопировать папку `bookshop.local` в `C:\OSPanel\domains\`
2. Добавь в файл `hosts` строку: 127.0.0.1 bookshop.local
3. В phpMyAdmin создать базу данных с именем `bookshop_db`
4. Импортировать в неё файл `bookshop_db.sql`
5. Открыть в браузере: `http://bookshop.local/`

Данные для входа: 

| Администратор | `admin@bookshop.local` | `admin123` | - авторизоваться как обычный пользователь для входа

Пользователей можно создать самостоятельно.

Технологии

- PHP 7.2
- MySQL
- HTML + CSS (FontAwesome)
- Open Server Panel

- Основные функции

- Каталог книг с фильтрацией по категории, автору и году
- Покупка и аренда книг на 2 недели, 1 месяц или 3 месяца
- Корзина покупок
- Админ-панель: управление книгами, заказами, напоминания
- Загрузка обложек книг через админку
