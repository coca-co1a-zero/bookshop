# bookshop
-
1. В phpMyAdmin создать базу данных с именем `bookshop_db`
2. Импортировать в неё файл `database.sql`
3. Открыть в браузере: `http://bookshop.local/`

 Данные для входа

| Администратор | `admin@bookshop.local` | `admin123` |
| Пользователь | Зарегистрировать самостоятельно | 

Технологии

- PHP 7.2
- MySQL
- HTML + CSS (FontAwesome)
- Open Server Panel

Основные функции

- Каталог книг с фильтрацией
- Покупка и аренда книг (2 недели, 1 месяц, 3 месяца)
- Корзина покупок
- Админ-панель: управление книгами, заказами, напоминания
- Загрузка обложек книг
